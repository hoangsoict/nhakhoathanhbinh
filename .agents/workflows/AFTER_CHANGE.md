# Workflow: After Change

1. Kiểm tra `git diff` các file đã sửa.
2. Chạy lint/build/test phù hợp khi feasible.
3. Cập nhật `.agents/memory/SESSION_STATE.md`.
4. Cập nhật `.agents/memory/CHANGELOG_AI.md`.
5. Cập nhật `.agents/memory/DECISIONS.md` nếu có quyết định mới.
6. Cập nhật `.agents/memory/OPEN_QUESTIONS.md` nếu còn điểm chưa rõ.
7. Trong final, nêu file đã sửa, lệnh kiểm tra, rủi ro còn lại.
