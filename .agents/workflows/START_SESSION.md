# Workflow: Start Session

1. Đọc `AGENTS.md`.
2. Đọc `.agents/PROJECT_INDEX.md`.
3. Đọc `.agents/ARCHITECTURE.md`.
4. Đọc `.agents/TAKEOVER.md`.
5. Đọc `.agents/memory/SESSION_STATE.md`.
6. Đọc `.agents/memory/DECISIONS.md`.
7. Chạy `git status`.
8. Chọn skill/rule liên quan đến task.
9. Nếu task không rõ, hỏi ngắn gọn hoặc ghi vào `OPEN_QUESTIONS.md` khi phù hợp.
